# esp-open-rtos-example

# update-spiffs   
Example which renews a file on SPIFFS via a network.   

	$ cd Your-esp-open-rtos-Home
	$ git clone https://github.com/nopnop2002/esp-open-rtos-example
	$ cd esp-open-rtos-example/update-spiffs
	$ make
	$ sudo make flash
