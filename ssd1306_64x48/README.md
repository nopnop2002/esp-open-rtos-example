# ssd1306-64x48

Display Japanese Font to ssd1306 OLED.   

__This example require 4MByte FLASH size.__   
__Before build program, Make erase all Flash.__   

---

# i2c Connection

|OLED(i2c)||ESP8266|
|:-:|:-:|:-:|
|GND|--|N/C|
|VDD|--|3.3V|
|SCK|--|GPIO5|
|SDA|--|GPIO4|
